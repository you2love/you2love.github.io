### 检查代码中是否包括密码密钥之类[gitleaks](https://github.com/zricethezav/gitleaks)

```bash
git clone https://github.com/zricethezav/gitleaks.git
cd gitleaks
make build
# 主要可用在阻挡提交
# detect protect
gitleaks detect -s=. -r=out.json
```
